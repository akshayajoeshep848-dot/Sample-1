ORDER

The order property controls the order in flex items appear within the flex container by default items are laid out in their source order(as they appear in the HTML)
 .Accept an integer value(+ve, -ve or zero)
 .Default value zero(0)
 .Items with lower order values appear first. Items with same order value are laid out in the source order
 .This property allows you to change the visual order without altering the HTML structure, Which can be useful
   for responsiveness or capability consideration(Though be mindful that order still matters for things like tab navigation)

Flex-grow

Felx-grow property defines the ability of a flex items to grow if necessary. It specifies what proportion of the remaining space in the flex container should be distributed to them.

  • accept a unitless non-negative numbers(eg, 0, 1, 2)
  • Default value is 0.(Meaning the item will not grow to full available soace).
  • If all items have flex-grow:1 ,the remaining space is distributed equally among them.
  • If one item has flex-grow: 2and others have flex-grow:1; The items with flex grow 2 will take up twice as much of the available space as the others.
This property is applied after the items have been sized according to flex-basis.It only has an effect if there is +ve free space in flex container.



Flex shrink

Flex property defines the ability of a flex item to shrink if necessary

  • Accept a unitless non-negative number.
  • Default value is 1.
  • A value of 0 means the item will not shrink,
  • .if one item has flex-shrink: 2, and others have flex-shrink:1, the item with flex-shrik: 2, will shrink twicw as much the others when space is insufficient. The amount an item shrink is also proportional to its flex-basis
