Functions are fundamental building blocks in js. They are reusabe blocks of code that perform a specific task.Today we'll explore how to define and use functions, focusing declarations and Function expressions.

What is Function

A function is a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

    Key benfits of using functions:
  • Reusability:Write code once and use it multiple times.
  • Modulity: Break down complex problems into smaller, managaeable pieces.
  • Readability: Make code easier to understand by giving descriptiev names to block of logic.
