Increment(++) and Decrement(--)

  • If let a = 5; a++;, what is the new value of a?
  • If let b = 10; b--;, what is the new value of b?
  • What is the value of let c = 7; c++ + 5;? (Hint: The value returned by c++ is the value before the increment)
  • What is the value of let d = 3; ++d + 5;
  • What is the final value of e after these operations: let e = 10; e--; e++;
