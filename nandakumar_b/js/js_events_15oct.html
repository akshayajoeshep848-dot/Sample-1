EVENT TYPES

    Event types
  • Mouse events: click, dbclick mousedown, mouseup, mousemove,mouseover, mouseout
  • Keyboard events: keydown, kwyup,keypress
  • Form events: submit, change, focus, blur
  • window events: load, resize, scroll, unload
  • Clipboard events: Copy, cut paste

Event Listners

      To respond to events. you attached event listeners(also called event handlers) to DOM elements
