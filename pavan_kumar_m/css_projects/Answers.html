Css ?

Answer
Definition

Css stands for cascading style sheets

It is used to design and style the webpages - like setting colors ,fonts,layouts,sizes and etc.

Usage

To make the websites look beautiful and userfriendly

To apply styles to multiple pages with one file

To separate content (HTML) from design (CSS)

Types of CSS:

1.Inline CSS :Writing the CSS directly inside an HTML tag using the style attribute.It applies the style to just one specific element.

Usage

For overriding the other css styles

For quick testing or small changes

<p style="color: red; font-size: 18px;">This is a red paragraph.</p>

2.Internal CSS :Writing the CSS styles inside <style> in the head section of the document.It applies the styles to single web page.

Usage

When our styles are only for one page

For small projects or simple web pages

<!DOCTYPE html> <html> <head> <style> h1 { color: blue; font-size: 30px; } p { color: gray; } </style> </head> <body> <h1>Hello!</h1> <p>This is a paragraph.</p> </body> </html>

3.External CSS :Writing the CSS code in seperate .css file. and linking it to our html file using the <link> tag in the <head> section

Advantages:

Code is clean and organized

Styles can be reused across many pages

Keeps HTML and CSS separate

What is font?

Answer
Definition

In CSS, font is used to control how the text looks on a web page. It includes the type of font (like Arial,sans-serif etc), size of the text, boldness of text, italic style, and spacing between lines etc.

Usage

We use css font properties to style the text to look better and readability

Example
p { font-family: 'Arial', sans-serif; font-size: 16px; font-weight: bold; font-style: italic; }

Font shorthand property?

Answer
Definition

Font short hand property is a shortcut used to set multiple font properties in a single line instead of writing the each line seperately

Usage

It saves time and makes the code cleaner.

Example
p { font: italic bold 16px/1.5 Arial, sans-serif; }

Font Family?

Answer
Definition

Font family is used to set the type of font to be displayed on webpage

Usage

We use font family to modify the existing text to new font styles

Example

p { font-family: "Times New Roman", Georgia, serif; }

This means:

First choice: Times New Roman

If not available: Georgia

If none available: use any serif font as a backup

Box model?

Answer
Definition

Box model defines that every html element is surrounded or wrapped by a rectangular box and made up of four layers

Box Model Layers (From Inside, Out):

  1. Content: The actual text or image inside the html element
  2. Padding: Space between the content and border(inside space)
  3. Border: A line which is visible around the content or padding
  4. Margin: Space outside of the border(space between this box and other boxes)
Example
div { width: 200px; padding: 10px; border: 5px solid black; margin: 20px; }

Padding Shorthand?

Answer
Definition

Padding shorthand prperty is used to apply the space between the content and border for all four sides (top,right,bottom,left) in single line.

How Values Work:

  1. padding:10px;--- All sides get 10px
  2. padding:10px 20px;--- Top & Bottom =10px, Left & right=20px
  3. padding:10px 20px,30px;--- Top=10px,Left & Right =20px,Bottom=30px
  4. padding:10px 20px 30px 40px;--- Top=10px,Right=20px,Bottom=30px,Left=40px

Note: It follows the clockwise rule: Top → Right → Bottom → Left

Example
p { padding: 5px 10px 15px 20px; }

Margin Shorthand?

Answer
Definition

Margin short hand property is used to set the space outside of an element's border on all four sides (top,right,bottom,left) in a single line

Use:

It helps to control the spacing between the elements on a webpage

How Values Work:

  1. margin: 10px; --- All sides get 10px
  2. margin: 10px 20px; --- Top & Bottom =10px, Left & right=20px
  3. margin: 10px 20px 30px; --- Top=10px,Left & Right =20px,Bottom=30px
  4. margin: 10px 20px 30px 40px; --- Top=10px,Right=20px,Bottom=30px,Left=40px

Note: It follows the clockwise rule: Top → Right → Bottom → Left

Example
p { margin: 5px 10px 15px 20px; }

Border Width?

Answer
Definition

Border width is used to set the thickness(width) of an element's border. We can control all the sides at once or each side individually. It works along with the border style(solid ,dotted,dashed etc) and border-color

How Values Work:

  1. border-width: 1px; --- All sides get 1px
  2. border-width: 1px 2px; --- Top & Bottom =1px, Left & right=2px
  3. border-width: 1px 2px 3px; --- Top=1px,Left & Right =2px,Bottom=3px
  4. border-width: 1px 2px 3px 4px; --- Top=1px,Right=2px,Bottom=3px,Left=4px
Accepted Units

px, em, rem, etc

Note: It follows the clockwise rule: Top → Right → Bottom → Left

Example
div { border-width: 4px; border-style: solid; }

Border Style?

Answer
Definition

Border Style is used to define the appearance of the border like whether it is dotted,dashed,double,inset,outset,ridge,none etc.

How Values Work:

  1. border-style: solid; --- All sides solid
  2. border-style: solid dashed; --- Top & Bottom: solid, Left & Right: dashed
  3. border-style: solid dotted double; --- Top: solid, Left & Right: dotted, Bottom: double
  4. border-style: solid dotted dashed double; --- Top=solid,Right=dotted,Bottom=dashed,Left=double
Accepted Units

px, em, rem, etc

Note: It follows the clockwise rule: Top → Right → Bottom → Left

Example
p { border-style: solid dotted dashed double; }

Border Color?

Answer
Definition

Border Color is used to set colour of an elemnt's border. It can apply same colour to all four sides or different colours for each side -top,right,bottom,left. To make the color visible, we must also define a border-style (like solid, dashed, etc.).

How Values Work:

  1. border-color: red; --- All 4 sides = red
  2. border-color: red green; --- Top & Bottom = red, Left & Right = green
  3. border-color: red green blue; --- Top → red, Left & Right = green, Bottom = blue
  4. border-color: red green blue black; --- Top = red, Right = green, Bottom = blue, Left = black
Accepted Units

px, em, rem, etc

Note: It follows the clockwise rule: Top → Right → Bottom → Left

Example
p { border-style: solid; border-width: 2px; border-color: red green blue yellow; }

Box Shadow ?

Answer
Definition

Box shadow is used to add the shadow effects to elements It creates a shadow behind a box (like a div, button, or card)

Example
selector { box-shadow: h-offset v-offset blur spread color; }
  1. h-offset -> Horizontal shadow offset (right if positive, left if negative)
  2. v-offset -> Vertical shadow offset (down if positive, up if negative)
  3. blur -> Blur radius
  4. spread -> Optional -> how much the shadow expands/shrinks
  5. color -> Shadow color (e.g. black, rgba(0,0,0,0.5))
  6. inset -> (Optional) places shadow inside the element

Units ?

Answer
Definition

CSS units define the measurement values used in styling properties like width, height, padding, margin, font-size, etc.

Types(Absolute - Fixed size)
  1. px(pixels)
  2. pt(points)
  3. cm(centimeter)
  4. mm(millimeter)
  5. inch(inches)
Types(Relative - flexible/responsive)
  1. em-relative font size to the parent element
  2. rem- relative font size to root element-(1 rem=16px)
  3. % - Relative To Parent element
  4. vh-viewport height
  5. vw-viewport width
  6. vmin
  7. vmax

px ?

Answer
Definition

px stands for pixels, and it is the most commonly used absolute CSS unit. 1px equals to one dot on the screen (based on the device resolution)

Example
p { font-size: 16px; margin: 20px; padding: 10px; }

pt ?

Answer
Definition

pt stands for points it is one of the absolute unit.

1 pt = 1/72 of an inch (approx. 0.35 mm)

Example
p { font-size: 12pt; }

rem ?

Answer
Definition

rem stands for root em and it is relative css unit that is scaled based on the root element's (<html>) font size

1 rem=font-size of the <html> element

Default browser font size=16px, so:

1rem = 16px

Example
html { font-size: 16px; }
p { font-size: 1.5rem; } /* = 24px (1.5 × 16px) */

em ?

Answer
Definition

em is a relative css unit used for font size ,spacing, and layout.

It is based on the font size of the element itself or its parent element

1em = current element's font size

If not set directly, it inherits from the parent’s font size

Example
body { font-size: 16px; } p { font-size: 1.5em; /* = 24px */ }

vh ?

Answer
Definition

vh stands for viewport height. It’s a relative CSS unit that refers to 1% of the height of the browser's viewport (visible area of the screen).

So:

100vh = 100% of the screen height

50vh = half the screen height

Example
section { height: 100vh; background-color: lightblue; }

vw ?

Answer
Definition

vw stands for viewport width. It is a relative CSS unit based on the width of the browser’s viewport (the visible part of the screen).

1vw = 1% of the viewport width

100vw = full width of the screen

Example
div { width: 50vw; background-color: orange; }

%(Percentage) ?

Answer
Definition

The % is a relative css unit relative to its parent's element.

It is responsive and adapts to the different screen sizes.

Example
.container { width: 80%; margin: 0 auto; }

min-width?

Answer
Definition

min-width is used to set the minimum width to an element.

Even if the screen becomes smaller,the element won't get smaller than the value we set as min-width

Example
.element { min-width: 300px; }

max-width?

Answer
Definition

max-width is used to set the maximum width to an element.

Even if the screen becomes wider,the element won't become bigger than the value we set as max-width

Example
.element { max-width: 600px; }

position?

Answer
Definition

position property is used to control how an element is placed on the web page

It tells the browser that where and how the element should appear by using top,right,bottom and left values

Values for position :
  1. static: Default Position
  2. relative: Moves the element relative to its original position
  3. absolute : Positions the element relative to its nearest positioned ancestor
  4. fixed : makes the element fixed on the screen even when scroling
  5. sticky : scrolls normally but sticks at a point when reaching a position
Example
.element { position: value; top: 10px; left: 20px; }

position-relative ?

Answer
Definition

Position relative means the element stays in its normal position ,but we can move it using top,right,bottom,left.

Example
.box { position: relative; top: 10px; left: 20px; }

The element moves 10px down and 20px to the right but its original space is reserved

Position-absolute?

Answer
Definition

Absolute allows you to place an element exactly where you want on the page or inside a container.

It is positioned relative to its nearest positioned ancestor (an element with position: relative, absolute, or fixed).

The element is removed from the normal layout flow and positioned using top, right, bottom, left

If no such ancestor exists, it’s positioned relative to the root element ()

Example
.box { position: absolute; top: 20px; left: 30px; }

position-fixed ?

Answer
Definition

Position fixed means the element fixed in its original position ,it stays same place on the screen even when the user scrolls.

It is always positioned relative to the viewport, not any container.

Example
.navbar { position: fixed; top: 0; left: 0; width: 100%; background-color: black; color: white; }

position-sticky ?

Answer
Definition

position: sticky(relative+fixed)

Position sticky means the element scrolls normally with page until it reaches to the certain position.

Then it sticks and stays fixed at that position even when scroling the page.

Example
.header { position: sticky; top: 0; background: white; padding: 10px; }

The element moves 10px down and 20px to the right but its original space is reserved
