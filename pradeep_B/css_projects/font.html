Box model

ANSWER

DEFINTION

The CSS box model describes how HTML elements are displayed as rectangular boxes, defining their size and spacing. It consists of four main parts: content, padding, border, and margin. Each part contributes to the total space an element occupies on a webpage.

USAGE

In CSS, the term "box model" is used when talking about design and layout.

CODE

width: 320px; height: 50px; padding: 10px; border: 5px solid gray; margin: 0;
