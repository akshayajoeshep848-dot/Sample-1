CSS Font-Styles

what is font?

Answer

Definition:

In CSS, a font refers to the style and appearance of text on a webpage. It is controlled using various font-related properties that define the typeface, size, weight, style, and line spacing.

usage

                You use font-related properties in CSS to:

                 1)Apply a specific typeface to text

                 2)Adjust the size and weight for visual hierarchy

                 3)Make text italic or normal

                 4)Control line spacing for better readability

example

p { font-family: 'Roboto', sans-serif; font-size: 16px; font-weight: 400; line-height: 1.5; }

font-family

Answer

Defination:

In CSS,the font-family property it used to define how the text to be displayed on the webpage

Usage;

               1) Controls the visual identity and readability of your website’s text.
               
                selector {
                   font-family: "Primary Font", "Fallback Font", generic-family;
                }
               
               2)Font names with spaces must be in quotes.

               3)Always end with a generic font family like sans-serif, serif, or monospace.

Example:

body { font-family: "Open Sans", "Helvetica Neue", Arial, sans-serif; } 
                
this means;

                  1)Try to load "Open Sans" first.
                  2)If not available, use "Helvetica Neue".
                  3)Then Arial.
                  4)Finally, if none are available, fall back to any sans-serif font.

font-size

Answer

Defination;

font-size is a CSS property used to specify the size of the text. It controls how big or small the font appears on the screen.

Usage;

         1)Controls readability and visual hierarchy.
         2)Helps maintain responsive design across devices.
         3)Essential for accessibility (users with vision issues).

Example;

h1 { font-size: 32px; } p { font-size: 1.2em; }

font-weight

Answer

Defination;

font-weight is a CSS property used to control the thickness (or boldness) of the text.

Usage;

                1)Emphasize important text (e.g., headings, buttons)
                2)Improve readability and UI structure
                3)Follow branding guidelines (light, regular, bold fonts)

Example;

h1 { font-weight: bold; /* or 700 */ } p { font-weight: normal; /* or 400 */ } .light-text { font-weight: 300; }

font-style

Answer

Defination;

The font-style property in CSS is used to specify the style of the font, such as normal, italic, or oblique.

Usage

                The most common use of font-style is to italicize text for emphasis, such as in quotes, citations, or emphasized words.

Example

p { font-style: italic; }

font-variant

Answer

Defination

"font-variant is a CSS property used to control the use of small-caps letters. It allows you to display lowercase text using uppercase letterforms, but at a smaller size."

Usage

It’s mostly used for typographic styling when you want text to appear formal or traditional, such as in headings, titles, or citations.

Example

h2 { font-variant: small-caps; }

letter-spacing

Answer

Defination

"letter-spacing is a CSS property used to control the horizontal space between characters (letters) in a text element. It helps improve readability or achieve specific design styles."

Usage

It is commonly used in headings, buttons, or stylized text to make the text appear more spaced out or condensed, depending on the visual need.

Example

h1 { letter-spacing: 2px; }

line-height

Answer

Defination

"line-height is a CSS property that controls the amount of vertical space between lines of text. It determines how tall each line box is, affecting the spacing between lines in a block of text."

Usage

It’s commonly used to improve readability, especially for paragraphs, and to make text layouts more visually balanced. A larger line-height makes text easier to read, especially on screens.

Example

p { font-size: 16px; line-height: 1.5; }

word-spacing

Answer

Defination

"word-spacing is a CSS property that controls the horizontal spacing between words in a block of text. It helps improve text readability or achieve specific design effects."

Usage

It's often used in headings, banners, or stylized text where adjusting space between words enhances appearance or layout clarity. You can increase or decrease spacing depending on the design need.

Example

h1 { word-spacing: 8px; }

font-shorthand

Answer

Defination

"font is a shorthand property in CSS that allows you to set multiple font-related properties in a single declaration. It combines values like font-style, font-variant, font-weight, font-size, line-height, and font-family."

Usage

This shorthand is useful for writing clean, compact code and applying all key font styles at once. It’s most commonly used for styling headings, paragraphs, and UI text elements.

Example

p { font: italic small-caps bold 16px/1.5 "Times New Roman", serif; }

text-decoration-line

Answer

Defination

"text-decoration-line is a CSS property that specifies what kind of line decoration is added to text — such as an underline, overline, or line-through. It is part of the text-decoration shorthand."

Usage

It's used to style text for emphasis or formatting, like underlining links, striking through completed tasks, or adding overlines for visual effect.

Example

h1{ text-decoration-line: none | underline | overline | line-through | multiple values; }

text-decoration-color

Answer

Defination

"text-decoration-color is a CSS property used to set the color of text decoration lines, such as underlines, overlines, or line-throughs. It allows you to customize the color of the decoration independently from the text color."

Usage

It's useful when you want to highlight or style text decorations (like underlines) in a different color than the text itself — for visual emphasis or design contrast.

Example

a { text-decoration-line: underline; text-decoration-color: red; }

text-decoration-style

Answer

Defination

"text-decoration-style is a CSS property that specifies the style of the text decoration line, such as underline, overline, or line-through. It allows you to change how the line appears — whether it's solid, dotted, dashed, wavy, or double."

Usage

It’s used along with text-decoration-line to control the visual appearance of text decorations. Designers often use it for creative effects or emphasis.

Example

h1 { text-decoration-line: underline; text-decoration-style: dashed; }

text-decoration-thickness

Answer

Defination

"text-decoration-thickness is a CSS property that sets the thickness (or width) of the text decoration line, such as underlines, overlines, or line-throughs. It allows designers to control how bold or thin the decoration line appears."

Usage

It is used to enhance visual emphasis, design consistency, or accessibility by making decoration lines more visible or stylized. You can set it to a fixed value or relative to the font size.

Example

a { text-decoration-line: underline; text-decoration-thickness: 3px; }

text-decoration-shorthand

Answer

Defination

"text-decoration is a CSS shorthand property that combines several text decoration-related properties into one declaration. It includes text-decoration-line, text-decoration-color, text-decoration-style, and optionally text-decoration-thickness."

Usage

It’s used to apply all aspects of text decoration (like underlining style, color, and thickness) in a single, clean line of CSS. This makes styling more efficient and easier to maintain.

Example

a{ text-decoration: [line] [style]? [color]? [thickness]? ; }
